Visualization of Rainfall Data for Hampton

Recorded at Bentleigh (086006) Weather Station

Srinivas Chakravarty
Developing Data Products Course Assignment

Background

The Australian Government Bureau of Meterology (BOM) maintains records of various aspects of climate data such as Rainfall, Temperature, and Wind etc.

The data is recorded at several locations through appropriately located Weather Stations.

BOM makes past data (currently from 1937 to 2006) freely available online to the general public.

Rainfall data for Hampton (Victoria), as recorded at Bentleigh (086006) Weather Station, has been chosen as a suitable candidate for visualization, with emphasis on providing a simplified interface for the end user to visualize the data in a concise format.

The user can specify the Month and the Cut-off Year in order to see the Rainfall pattern for THAT Month across years, from THAT Cut-off Year onwards.

Technical Details

The Shiny App for this project consists of R-code in the ui.R and server.R components.

Component ui.R provides the user a Drop-Down box from which the Month of Interest can be selected. The Cut-off Year of Interest is selected on a Slider Bar.

Default values are provided for both the Month and the Year of Interest.

Component server.R reads in the Rainfall CSV data. It maps Month Names to Month Numbers, and interactively extracts ALL available information for THE SELECTED Month from THE SELECTED Year onwards.

The extracted data is presented as a Graph which changes dynamically as the selections are changed.

The end user does NOT need to understand the implementation of the underlying technical details in order to use the App.
